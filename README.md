### Capítulo 1: Introducción a Git
Aprende qué es Git, por qué es importante y cómo instalarlo en tu sistema. Descubre los conceptos fundamentales como repositorios, commits y ramas.

### Capítulo 2: Flujo de trabajo básico
Domina los comandos esenciales para trabajar con Git día a día. Incluye creación de repositorios, staging, commits y visualización del historial.

### Capítulo 3: Repositorios remotos
Trabaja con GitHub, GitLab y otros servicios de hosting. Aprende a colaborar con otros desarrolladores usando push, pull y resolución de conflictos.
